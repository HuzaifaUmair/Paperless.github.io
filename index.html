Introduction

The main purpose of this project is to develop a paperless parking system using an app, website and hardware. This smart parking app will be used normally by parking manager or employee (2). Today parking system are necessary to shifting from manual to online system. Presently conventional cameras or sensor located in different places of the parking area use to detect parking details. it is difficult to carry token of vehicles given by the parking boy to everyone. We want to develop a system in which we detect vehicle plate number by using cameras/sensor from the entrance gate. After that, information of number plate is converted into number and store in database and update information on user app. It tells us the time of entrance and exit of vehicle. When user leave the parking then also update data on database and user app. The registration of vehicle is done at parking system. There are two cameras or sensor one for entrance and second for leaving parking area. Both camera/sensors update the entrance and leaving state of vehicles and also update on User app. The payment method is on monthly bases. User can check availability of slots by using his Mobile application. The main purpose of the project is to reduce time and paper consumption. The paperless parking system is step toward modern technology that can be implemented on parking Places. The paperless parking system improve environmentally friendly, Ease of access and use, Resident satisfaction, Financial beneficial for User and Owner, Modernization and future reliability
